# The Matter at Hand

A CHESS PAWN…

IN A PUZZLE GAME…

ABOUT CARDS…?


## Controls:
Arrow Keys - Select Card / Move

Space / Up - Use Card

R - Restart Level

Escape - Discard

## Tools Used:
Eclipse + Self-Built Java Engine

Audacity

Aseprite

Casio PX-150 Keyboard

Endesga 32 Color Pallete

Average completion time: 10-15 minutes (15 levels in total)
